2. インストール¶

2.1. 必要なパッケージ¶

  • Python (3.6 - 3.7)

  • gcc-c++ (4.8.5 - 9.3.0)

  • bzip2

  • fcgi (オプショナル)

  • mysql, postgresql (オプショナル)

  • msgpack (オプショナル)

  • gtest (オプショナル)

2.2. 手順¶

$ ./waf configure
$ ./waf build
$ sudo ./waf install

その他オプションは ./waf --help を参照のこと。
