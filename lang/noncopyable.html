3.3.7. pfi::lang::noncopyable¶

3.3.7.1. 概要¶

コピー不可なクラスを作るシンプルな方法を提供する。

3.3.7.2. 使い方¶

class noncopyable

このクラスを、private継承してクラスを作る。

3.3.7.3. サンプルコード¶

class A : noncopyable{ // private継承させる
};

int main()
{
  A a, b;
  a=b; // compile error!
}
