3.3.3. pfi::lang::cast¶

3.3.3.1. 概要¶

C++的なキャストを用意。

3.3.3.2. lexical_cast¶

template <class T, class U>
T lexical_cast(const U &r);

T型への字句的キャスト。 実装は、stringstreamに突っ込んでstringstreamから取り出しているだけ。

string num("123");
cout<<lexical_cast<int>(num)*2<<endl; // 246
